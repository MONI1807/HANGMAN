# 🎮 Hangman Game

## 📌 Project Overview

The Hangman Game is a console-based Python application developed as part of the CodeAlpha Python Programming Internship. The game challenges players to guess a hidden word one letter at a time using a provided clue. Players must identify the word before exhausting their limited number of incorrect guesses.

This project demonstrates fundamental Python programming concepts and showcases problem-solving skills through an interactive text-based game.

---

## ✨ Features

* Random word selection
* Clue-based gameplay
* ASCII Hangman visualization
* Input validation
* Win/Loss score tracking
* Replay functionality
* Exit option during gameplay
* User-friendly console interface

## 📋 Task Information

### Task 1: Hangman Game

**Internship:** CodeAlpha Python Programming Internship

**Objective:**
Develop a text-based Hangman game where players guess a secret word one letter at a time.

### Requirements

* Use a predefined list of words
* Select a random word for each game
* Allow letter-by-letter guessing
* Limit incorrect guesses to six attempts
* Display the current game state
* Implement win and lose conditions
* Provide user-friendly interaction
